# Referencia de Comandos

Lista completa de comandos DBL disponibles.

## Setup

Comandos para inicializar y configurar.

- [`dbl init`](setup/init.md) - Inicializar nuevo proyecto
- [`dbl update`](setup/update.md) - Actualizar DBL
- [`dbl validate`](setup/validate.md) - Validar esquema
- [`dbl version`](setup/version.md) - Ver versión

## Sandbox

Trabajar con sandbox de desarrollo seguro.

- [`dbl sandbox start`](sandbox/start.md) - Crear sandbox
- [`dbl sandbox apply`](sandbox/apply.md) - Aplicar cambios a principal
- [`dbl sandbox rollback`](sandbox/rollback.md) - Descartar sandbox
- [`dbl sandbox status`](sandbox/status.md) - Ver estado sandbox

## Cambios

Gestionar cambios en base de datos.

- [`dbl diff`](changes/diff.md) - Ver cambios sin guardar
- [`dbl commit`](changes/commit.md) - Guardar cambios como capa
- [`dbl reset`](changes/reset.md) - Reconstruir desde capas

## Historial

Ver y navegar por historial de cambios.

- [`dbl log`](history/log.md) - Ver historial de capas

## Ramas

Gestionar ramas para desarrollo paralelo.

- [`dbl branch`](branching/index.md) - Crear/gestionar ramas
